# Snake

Un petit jeu Snake écrit avec [Pygame](https://www.pygame.org/) pour jouer directement depuis le bureau.

## Installation

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Utilisation

```bash
python snake.py
```

Commandes :
- Flèches : déplacer le serpent.
- R : recommencer après un game over.
- Échap ou fermer la fenêtre : quitter.

## Développement

Pour vérifier la syntaxe avant de lancer le jeu :

```bash
python -m compileall snake.py
```
